Aims & objectives
  • Main aim. By the end, pupils will be better able to name football actions & answer about ability — Can you kick? — Yes, I can! Pass! Shoot! Score! Come on, come on! — with the verbs kick · pass · shoot · save · catch · throw · head · score.
  • Subsidiary. Remember-time cumulative recycle of earlier levels (incl. languages L2); folder worksheet (put-in-order — по порядку: kick → pass → shoot → score); a PE-style action relay.
  • Personal. Bilingual L2/L1; big safe movements, mime not real balls indoors; every child moves & shouts a verb every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • mime & say kick · pass · shoot · catch · save · score;
  • ask & answer Can you kick? — Yes, I can!;
  • chant the cheer Come on, come on!;
  • follow action calls in an order;
  • recycle Can you speak English? — Yes, I can!

Pron. watch
sharp one-syllable shouts; Score! long & proud; Come on! as a rhythmic cheer. Mime only indoors — no real kicking.
Instruction checks (kept tiny) · Assumptions
CCQ: (mime a kick) “Kick or catch?” → kick. (mime hands) → catch.
ICQ: “Real ball or PRETEND ball?” → pretend! Big careful moves.

Assumptions: children had Adventure→Fantastic L2; know Can you…? & action verbs (superhero). Football verbs are new but action-rich — mime carries meaning. Near-zero English + L1 prompt. Non-readers. Group = 12. Space cleared for safe movement.

Lesson framework — Oksana's real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Cumulative Q&A recycle lives at the start (Remember time); the lesson ends on a topic game, then the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), “choose the greeting → jump/run/swim/fly to your chair”.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives from the bus-box & shakes hands with each child (Hello! How are you?).
  • Remember time ⚡ — cumulative review of EVERYTHING so far: Q&A chain + a quick recall game (Criss Cross / board app).
  • Focus time ⚡ — presentation from the magic bus-box, topic song, board app, 2–3 active games.
  • Table time 🪑 — own folder worksheet, a different type each lesson, always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today's language, ending on a high note.
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
