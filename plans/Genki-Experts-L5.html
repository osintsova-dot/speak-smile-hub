🦝 GenkiEnglish Preschool · Experts · Lesson 5 — Rocket Launch! Numbers 10–100
Genki preschoolers (~5–6 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Experts level · Lesson 5 of 7✨ count by tens 10–100 · countdown · blast off!12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to count up in tens 10–100 and count down to blast off — Ten, twenty, thirty… one hundred! — Ten, nine… blast off! — jumping higher with each ten.
  • Subsidiary. Cumulative Remember-time recycle of ALL earlier topics (incl. Experts L2–L3: How many?, 13–20); folder mini-craft (build & launch a rocket); our big-book countdown story.
  • Personal. Bilingual L2/L1; count by tens as a chant with a jump per ten; every child counts & launches; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • count up in tens ten… one hundred!;
  • count down ten, nine… blast off!;
  • join in the rocket chant with jumps;
  • build & “launch” a paper rocket;
  • lead a countdown for the class.

Chunks
Ten, twenty, thirty… one hundred! · Ten, nine, eight… blast off! · countdown  ·  (recycle) How many? · Count to twenty!
TPR gesture
each ten a bigger crouch-and-jump · one hundred arms wide · countdown fingers down 10→1 · blast off leap up.
Pron. watch
keep the -ty ending clear (thir-TY, not thir-TEEN); rhythmic, one beat per ten. Countdown said crisply, getting quieter then a big BLAST OFF!
Instruction checks (kept tiny) · Assumptions
ICQ: “Counting UP or DOWN to blast off?” → down.
ICQ: “After forty comes…?” → fifty.

Assumptions: children count 1–20 (Experts L2–L3); tens are new but chant-friendly. Value of “fifty” is abstract at 5 — it’s a rhythmic chant with big jumps, not maths. Near-zero English + L1 prompt. Non-readers — tens shown as rocket-stage pictures. Group = 12.

Classroom management — signals
“Class? — Yes!” — you: “Class!” → children stop & echo “Yes!”
Volume control 👋 — raise arm = louder, lower = quieter → instant settler.
“Speaky says…” — do the action only when Speaky the raccoon says it.
“Freeze!” / statue — everyone becomes a statue, mid-movement reset.
Countdown — “10… 9… 8…” with fingers shown (perfect for today!).
“Come with me!” 🚊 — line up as a train (паровозик) to change zone.
Lesson framework — Oksana’s real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Cumulative Q&A recycle lives at the start (Remember time); the lesson ends on a high-note game on today’s language.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), “choose the greeting → jump/run/swim/fly to your chair”.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives from the bus-box & shakes hands with each child (Hello! How are you?).
  • Remember time — cumulative review of ALL topics with A4 flashcards / What’s missing? / app on the board.
  • Focus time — presentation from the magic bus-box, topic song, board app, 2–3 active games.
  • Table time 🪑 — own folder worksheets, a different type each lesson (listen & tick / colour by command / trace / match соедини / draw & tell / put-in-order / mini-craft), always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today’s language, so the lesson ends on a high note (kids don’t want to leave).
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
