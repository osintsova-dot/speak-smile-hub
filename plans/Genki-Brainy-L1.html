🦝 GenkiEnglish Preschool · Brainy · Lesson 1 — Numbers 1–6 — How many?
Genki preschoolers (~5–6 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Brainy level · Lesson 1 of 11✨ count 1–6 · How many?12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to count 1–6 and answer How many? — Six! — naming a small quantity on fingers & cards.
  • Subsidiary. Remember-time recycle of names & How are you?; the S&S big-book; a dot-to-dot folder sheet (connect the numbers).
  • Personal. Bilingual L2/L1; numbers always in context (How many? — Six!, not a bare list); every child jumps/shows fingers every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • count 1–6 aloud;
  • show a number on fingers;
  • answer How many? — Six!;
  • jump to the right number card;
  • connect the dots 1–6 & name them.

TPR gesture
each number = fingers held up; count tap each item; How many? open palms/shrug.
Pron. watch
three /θriː/ (tongue out, not “free”); six crisp /-ks/; keep counting a happy beat.
Instruction checks (kept tiny) · Assumptions
ICQ: “Do we JUMP or watch?” → jump.
ICQ: “Count AND say the number?” → yes.

Assumptions: children finished the Adventure level (names · feelings · commands) but are near-zero English for numbers → meaning from fingers, cards, music & the L1 prompt. Non-readers/non-writers. Attention 5–7 min. Group = 12.

Anticipated problems & solutions
Likely problemWhat you do
Numbers muddle upAlways tie a number to a QUANTITY (fingers + cards), never a bare recital. Count slowly, 3×.
“three” → “free”Model tongue-tip; over-do it playfully; the song carries it.
Jumping → over-excitement🧘 settler after each ⚡; Freeze! + volume-control arm.
A child can’t count yetWhole-group choral first; count WITH them, hand over hand on fingers.
Transitions get chaoticTrain: Come with me! — one signal, one destination.
Classroom management — signals
“Class? — Yes!” — you: “Class!” → children stop & echo “Yes!”
Volume control 👋 — raise arm = louder, lower = quieter → instant settler.
“Speaky says…” — do the action only when Speaky the raccoon says it.
“Freeze!” / statue — everyone becomes a statue, mid-movement reset.
Countdown — “5… 4… 3… 2… 1…” with fingers shown.
“Come with me!” 🚊 — line up as a train (паровозик) to change zone.
Lesson framework — Oksana’s real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Finale = cumulative Q&A chain + the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello / Good Morning song, “choose the greeting → jump/run/swim/fly to your chair”.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives on the bus-box & shakes hands with each child.
  • Remember time — cumulative review (thumb-up/down Q&A like a game) with A4 flashcards / memory / app on the board.
  • Focus time — presentation from the magic bus-box, topic song, board app, one active game.
  • Table time 🪑 — own folder worksheets, a different type each lesson (dot-to-dot / count & colour / craft / listen & tick / spinner / match / draw & tell / sort & stick), always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Remember time 2 + Closing — run the growing Q&A chain, Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
