🦝 GenkiEnglish Preschool · Brainy · Lesson 1 — Numbers 1–6
Genki preschoolers (~5–6 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Brainy level · Lesson 1 of 10✨ numbers 1–6 · How many?12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to count 1–6 and answer How many? — pointing and saying one, two, three, four, five, six in context (fingers, objects, claps), always counting something, never a bare list.
  • Subsidiary. Remember-time recycle of all Adventure (name · how are you · I can · left/right · pronouns · thank you); folder worksheet (trace & count); one-to-one counting habit.
  • Personal. Bilingual L2/L1; model finger-count facing the same way; accept mis-counts, re-count together; every child counts/speaks every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • count 1–6 on fingers & objects;
  • answer How many? — Five!;
  • join the Numbers song with gestures;
  • touch-count one item per number (one-to-one);
  • trace & count 1–6 on the worksheet.

Lesson framework — Oksana's real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🟢 / tables 🪑 / board 📺 (Genki app). The cumulative recycle lives at the start (Remember time); the finale is a fast game on today's language + the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), "choose the greeting → jump/run/fly to your chair".
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives from the bus-box & shakes hands with each child (Hello! How are you?).
  • Remember time — cumulative Q&A chain of ALL earlier topics + one review game.
  • Focus time — presentation from the magic bus-box, topic song, board app, active games.
  • Table time 🪑 — own folder worksheet, a different type each lesson, always language-linked (listen → do → say).
  • Reading time 🟢 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today's language, ending on a high note.
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
