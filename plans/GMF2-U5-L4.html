Aims & objectives
  • Main aim. By the end, learners will be better able to use 8 adjectives in opposite pairs (tall–short · fat–thin · clean–dirty · old–new) to describe snowmen and things: “He's a (tall) snowman. / He's got a (dirty) hat.”
  • Subsidiary. Song (CD2 45); adjective ORDER (a tall snowman — not a snowman tall); listen & tick discrimination (CD2 46 format); design-a-snowman description task.
  • Personal. Opposites live in the body (tall stretch / short crouch…); insist gently on adjective-before-noun — Russian order fights back.
By the end, pupils can…
  • say the 4 opposite pairs with gestures;
  • describe: “He's a fat, clean snowman. He's got a new hat.”;
  • draw from a partner's description.

Concept checks (CCQs) · Assumptions
“Snowman tall or TALL snowman?” → tall snowman — adjective first!
“Tall — show me! Short — show me!” → bodies stretch/crouch.

Assumptions: Story descriptions (fat tummy) seeded this; got-language returns from U3. Opposite pairs make 8 words learnable as 4 ideas. Группы: команда — максимум 4; при другой численности делим на команды по 3–4 (12=3×4 · 10=3+3+4 · 9=3×3 · 8=2×4 · 7=3+4).

Presentation · 8 adjectives
carpet

Opposite pairs with bodies. tall↑–short↓ · fat○–thin| · clean✨–dirty🧤 · old🧓–new🎁. Choral ×3 each pair with gestures; flash-test: you say one — class does the OPPOSITE (thinking twist!).

Order train. Card wagons: [a] [tall] [snowman] — build 4 sentences physically on the board ledge; class reads each. Wrong-order version shown → class fixes it.

▶ Take Off–Touch Down: Stand up if it's about SIZE… tall! …dirty! — categorising on the fly.

💪 Support: gesture + word. ⭐ Extension: two adjectives chained: a tall, thin snowman.

27–37′
Design-a-snowman dictation
desks, pairs

Model. You dictate, all draw (60 sec): He's a fat snowman. He's got a dirty hat and a long scarf! Reveal & laugh; checklist chorus back.

Pair round. A describes a secret snowman card, B draws; compare with the card — how close? Swap.

✔ Assess: the drawings check adjective comprehension; the descriptions check production — note both.

💪 Support: describe from the card's caption text. ⭐ Extension: add a they-line: They're playing with the snowman!
