Aims & objectives
  • Main aim. By the end, learners will be better able to ask about a town: “Is there a (bank)? — Yes, there is. / No, there isn't. · Are there any (museums)? — Yes, there are. / No, there aren't.”
  • Subsidiary. Understand the town dialogue (CD3 47); secret-map information gap (the unit's best game!); ordering questions (AB format); count-first ritual extended to questions.
  • Personal. The 2×2 grid from U7 grows a QUESTION row — show continuity, kids love “we know half already”; info-gap pairs must hide maps like poker players.
By the end, pupils can…
  • form both questions correctly by counting;
  • give all four short answers;
  • win an information-gap map game.

Structure
Is there a (bank)? — Yes, there is. / No, there isn't.  ·  Are there any (car parks)? — Yes, there are. / No, there aren't.
Form
Statement flips: There is → Is there…? · There are → Are there any…? — the flip gesture (turn hands over).
Pron. watch
Is_there /ɪzðeə/ linked; any weak; short answers land the stress: Yes, there IS.
Concept checks (CCQs) · Assumptions
“One bank — Is there or Are there?” → Is there a…!
“In the question with many — which little word?” → any!

Assumptions: There-statements automatic (U7) — the flip to questions is mechanical + gesture-supported. Town places active from L1. Info-gap format is new — model it theatrically. Группы: команда — максимум 4; при другой численности делим на команды по 3–4 (12=3×4 · 10=3+3+4 · 9=3×3 · 8=2×4 · 7=3+4).

Model theatrically with a strong pupil: folders up, poker faces: Is there a bank in your town? — Yes, there is! Are there any car parks? — No, there aren't! Tick the guesses.

▶ Info-gap pairs. Each child gets a secret town-map card (6 places, different sets). Find your partner's 6 by questions ONLY; mark a mini-grid. First pair done wins — but full answers only score.
