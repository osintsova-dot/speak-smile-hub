Aims & objectives
  • Main aim. By the end, learners will be better able to describe a home with “There's a (big kitchen). / There are (two bedrooms). / There isn't a (garage). / There aren't any (gardens).”
  • Subsidiary. Understand the house dialogue (CD3 25); singular/plural doors (echo of is/are from U2!); dream-house drawing dictation; ordering scrambled sentences (AB format).
  • Personal. Four forms is a lot — teach as 2×2 grid with colour; negatives get the fun (no dragons in my flat!).
By the end, pupils can…
  • choose There's/There are by counting;
  • make negatives with isn't/aren't any;
  • describe a pictured home in 3–4 sentences.

Structure
There's a (kitchen). / There are (two bathrooms).  ·  There isn't a (garage). / There aren't any (gardens).
Form
2×2 grid: one ✚ There's / many ✚ There are / one ✖ There isn't a / many ✖ There aren't any. any lives in the plural-negative box.
Pron. watch
There's /ðeəz/ one beat; aren't /ɑːnt/; any weak /əni/.
Concept checks (CCQs) · Assumptions
“Two bedrooms — There's or There are?” → There are (count!).
“No gardens at all — which box?” → There aren't ANY gardens.

Assumptions: Chunk There's-a seeded in L1; is/are counting logic from U2 transfers. Room words active. The 2×2 grid visual carries the lesson. Группы: команда — максимум 4; при другой численности делим на команды по 3–4 (12=3×4 · 10=3+3+4 · 9=3×3 · 8=2×4 · 7=3+4).
