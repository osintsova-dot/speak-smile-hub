🦝 GenkiEnglish Preschool · Adventure · Lesson 2 — How are you?
Genki preschoolers (~3.5–5 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Adventure level · Lesson 2 of 7✨ feelings · How are you?12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to ask and answer How are you? and say a feeling — I’m happy / sad / tired / hungry / cold / great — with a matching face and gesture.
  • Subsidiary. Remember-time recycle of L1 (name · RPS · commands); folder worksheet (listen & tick); feeling mimes.
  • Personal. Bilingual L2/L1; mirror faces; every child answers every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • ask How are you?;
  • answer with I’m happy / sad / tired / hungry / cold / great;
  • match a face/gesture to each feeling;
  • use the feeling to move in the Monster Game.

TPR gesture
happy smile+thumbs · sad rub eyes · tired yawn+stretch · hungry rub tummy · cold shiver+hug · great two thumbs up.
Pron. watch
keep I’m light and linked (I’m-happy); big feeling in the voice, not perfect vowels.
Instruction checks (kept tiny) · Assumptions
ICQ: “In the Monster Game, do we push or say a feeling?” → say a feeling.
ICQ: “How are you? — we answer with…?” → I’m happy / sad / tired…

Assumptions: Children had L1 (name, RPS, commands) — recycled at the start. Near-zero English + L1 prompt. Non-readers. Group = 12. Feelings shown by face + gesture, not defined.

Anticipated problems & solutions
Likely problemWhat you do
A feeling word won’t stickAnchor each to one clear gesture; over-use the face; the song and Karuta repeat it.
Monster Game gets too wildWalk first, then jog; Freeze! = brakes; two safe walls; the monster only tags gently.
Children move but stay silentNo feeling word = go back and try; the monster “can’t hear” until they say it.
Everyone shouts one feelingSpeaky assigns different faces; Karuta forces listening for the right one.
Over-excitement after ⚡🧘 settler; volume-control arm; countdown.
Classroom management — signals
“Class? — Yes!” — you: “Class!” → children stop & echo “Yes!”
Volume control 👋 — raise arm = louder, lower = quieter → instant settler.
“Speaky says…” — do the action only when Speaky the raccoon says it.
“Freeze!” / statue — everyone becomes a statue, mid-movement reset.
Countdown — “5… 4… 3… 2… 1…” with fingers shown.
“Come with me!” 🚊 — line up as a train (паровозик) to change zone.
Lesson framework — Oksana’s real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Cumulative Q&A recycle lives at the start (Remember time); the final game is on today’s language.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), choose the greeting → move to your chair.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives & shakes hands with each child.
  • Remember time — cumulative review with A4 flashcards / memory / bridge game / app on the board.
  • Focus time — presentation from the magic bus-box, topic song, board app, active games.
  • Table time 🪑 — own folder worksheets, a different type each lesson (listen & tick / colour by command / trace / match / draw & tell / put-in-order / mini-craft), always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today’s language, ending on a high note (kids don’t want to leave).
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
