🦝 GenkiEnglish Preschool · Challenge · Lesson 1 — I like animals
Genki preschoolers (~3.5–5 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Challenge level · Lesson 1 of 7✨ I like / I don't like + zoo animals12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to say what animals they like / don't like — I like kangaroos! · I don't like bears. — and ask a friend Do you like giraffes? with a thumbs-up / thumbs-down answer, using new zoo animals: kangaroos · bears · giraffes · penguins.
  • Subsidiary. Remember-time recycle of ALL learned so far (first Challenge lesson — recycle Adventure + Brainy: greetings · How are you? · numbers · Do you like food/animals?). Folder worksheet: listen & sort (happy face / sad face). Genki big-book.
  • Personal. Bilingual L2/L1; model like/don't-like with big face + thumb; accept one-word answers; every child speaks & moves every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • name 4 new animals: kangaroo · bear · giraffe · penguin;
  • say I like…! / I don't like… with a clear thumb;
  • ask & answer Do you like…? — Yes, I do! / No, I don't!;
  • sort animals into like / don't-like on the worksheet;
  • join the I like animals song with gestures.

TPR gesture

Pron. watch
plural -s is a soft hiss (kangaroos) — don't drill it; giraffe = "ji-RAAF"; keep like / don't like short with the thumb.
Instruction checks (kept tiny) · Assumptions
ICQ: "Thumbs UP — like or don't like?" → like.
ICQ: "Do we SAY the animal or stay quiet?" → say it.

Assumptions: children had Adventure + Brainy (they know Do you like…? with food & some animals). New = zoo animals + firmer I like / I don't like. Near-zero English + L1 prompt. Non-readers — all by ear + picture. Group = 12.

Classroom management — signals
"Class? — Yes!" — you: "Class!" → children stop & echo "Yes!"
Volume control 🙋 — raise arm = louder, lower = quieter → instant settler.
"Speaky says…" — do the action only when Speaky the raccoon says it.
"Freeze!" / statue — everyone becomes a statue, mid-movement reset.
Countdown — "5… 4… 3… 2… 1…" with fingers shown.
"Come with me!" 🚌 — line up as a train (паровозик) to change zone.
Lesson framework — Oksana's real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🟢 / tables 🪑 / board 📺 (Genki app). The whole cumulative Q&A recycle lives at the start (Remember time); the finale is a fast game on TODAY's language + the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), "choose the greeting → jump/run/swim/fly to your chair".
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives from the bus-box & shakes hands with each child (Hello! How are you?).
  • Remember time — cumulative Q&A chain across ALL learned topics + one review game (board app / flashcards).
  • Focus time — presentation from the magic bus-box, topic song, board app, 2–3 active games on today's language.
  • Table time 🪑 — own folder worksheet, a different type each lesson (listen & tick / colour by command / trace / match / draw & tell / put-in-order / mini-craft), always language-linked (listen → do → say).
  • Reading time 🟢 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today's language, ending on a high note (kids don't want to leave).
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
