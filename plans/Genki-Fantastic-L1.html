Fantastic level · Lesson 1 of 6✨ fruit + What's this? / What's that?12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to ask & answer about fruit — What's this? What's that? — It's an apple. It's a pineapple. — naming fruit (apple · orange · banana · pineapple · lemon · cherry · strawberry · watermelon).
  • Subsidiary. Remember-time cumulative recycle of all earlier levels (greetings · numbers · food · shapes · money); folder worksheet (match — соедини); the S&S fruit-market role-play.
  • Personal. Bilingual L2/L1; model a / an but never over-correct; every child names a fruit every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • ask What's this? / What's that? pointing near & far;
  • answer It's a(n) … with 6–8 fruit;
  • recognise fruit flashcards by ear;
  • play the market: What's this? — It's a banana!;
  • recycle Do you like apples? — Yes, I do!

TPR gesture
this = hold the fruit close to chest · that = point far across the room · yum = rub tummy / mime a bite.
Pron. watch
an apple / an orange (vowel) vs a banana; pineapple = 3 beats (PINE-ap-ple); watermelon long. Accept It's apple — meaning first.
Instruction checks (kept tiny) · Assumptions
CCQ: (hold fruit near) “This or that?” → this. (point far) → that.
ICQ: “Do we ASK first or ANSWER first?” → ask What's this?, then answer.

Assumptions: children had Adventure→Experts (know Do you like…?, colours, numbers). Fruit words are new; keep 6 core today (apple/orange/banana/pineapple/lemon/strawberry), the rest = exposure. a/an is hard — model it, don't drill. Near-zero English + L1 prompt. Non-readers. Group = 12.

Volume control 🖐 — raise arm = louder, lower = quieter → instant settler.
“Speaky says…” — do the action only when Speaky the raccoon says it.
“Freeze!” / statue — everyone becomes a statue, mid-movement reset.
Countdown — “5… 4… 3… 2… 1…” with fingers shown.

Lesson framework — Oksana's real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Cumulative Q&A recycle lives at the start (Remember time); the lesson ends on a topic game, then the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), “choose the greeting → jump/run/swim/fly to your chair”.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives from the bus-box & shakes hands with each child (Hello! How are you?).
  • Remember time ⚡ — cumulative review of EVERYTHING so far: Q&A chain + a quick recall game (What's missing? / board app).
  • Focus time ⚡ — presentation from the magic bus-box, topic song, board app, 2–3 active games.
  • Table time 🪑 — own folder worksheet, a different type each lesson, always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today's language, ending on a high note.
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
