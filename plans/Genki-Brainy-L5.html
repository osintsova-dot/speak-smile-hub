🦝 GenkiEnglish Preschool · Brainy · Lesson 5 — What's the weather? (part 1)
Genki preschoolers (~5–6 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Brainy level · Lesson 5 of 10✨ What's the weather? · sunny/rainy/windy12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to ask & answer What's the weather? with It's sunny / rainy / cloudy / windy — reading the sky & doing a weather action for each.
  • Subsidiary. Remember-time recycle of Adventure + numbers + age; folder worksheet (listen & tick today's weather); a weather-action habit (TPR).
  • Personal. Bilingual L2/L1; model facing the same way; accept fuzzy attempts and re-do together; every child speaks/moves every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • answer What's the weather? — It's sunny!;
  • recognise sunny/rainy/cloudy/windy;
  • do a TPR action for each weather;
  • ask about the weather;
  • listen & tick the weather on the sheet.

TPR gesture

Pron. watch
It's quick; the -y ending clear (sunn-y). Match the action to the word so meaning sticks.
Instruction checks (kept tiny) · Assumptions
ICQ: "Sunny — do we shiver or make a sun?" → make a sun.
ICQ: "Look out of the window — what's the weather TODAY?" → answer.

Assumptions: children have numbers & age. Weather is new vocab — anchor each word to one action + a picture card. Near-zero English + L1 prompt. Non-readers. Group = 12.

"Freeze!" / statue — everyone becomes a statue, mid-movement reset.
Countdown — "5… 4… 3… 2… 1…" with fingers shown.

Lesson framework — Oksana's real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🟢 / tables 🪑 / board 📺 (Genki app). The cumulative recycle lives at the start (Remember time); the finale is a fast game on today's language + the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), "choose the greeting → jump/run/fly to your chair".
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives from the bus-box & shakes hands with each child (Hello! How are you?).
  • Remember time — cumulative Q&A chain of ALL earlier topics + one review game.
  • Focus time — presentation from the magic bus-box, topic song, board app, active games.
  • Table time 🪑 — own folder worksheet, a different type each lesson, always language-linked (listen → do → say).
  • Reading time 🟢 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today's language, ending on a high note.
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
