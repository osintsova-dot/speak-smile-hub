🦝 GenkiEnglish Preschool · High · Lesson 4 — How did you get here? (Transport)
Genki preschoolers (~5–6 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
High level · Lesson 4 of 8✨ transport + I came here by…12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to name transport and say how they travelled — How did you get here? — I came here by car / by train / by rocket!
  • Subsidiary. Remember-time cumulative recycle of the whole year incl. months; folder worksheet (draw & tell — how you got here).
  • Personal. Bilingual L2/L1; each transport = one big mime + sound; accept By car! before the full chunk; every child moves/speaks every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • name transport: aeroplane · car · train · rocket · helicopter · boat · bus · mountain bike;
  • answer How did you get here? — I came here by…;
  • ask a friend How did you get here?;
  • mime + sound each vehicle.

Instruction checks (kept tiny) · Assumptions
ICQ: “Mime the vehicle AND say it?” → yes.
ICQ: (Monster Game) “Cross when the card matches HOW you came — run or push?” → run, no pushing.

Assumptions: children have the whole year + months. Most actually came by car/bus/on foot → we can use their real answer & add silly ones (rocket!). Non-readers → picture cards + sounds. Group = 12.

Lesson framework — Oksana's real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🟢 / tables 🪑 / board 📺 (Genki app). The whole cumulative Q&A recycle lives at the start (Remember time); the finale is a fast game on today's language + the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), “choose the greeting → move to your chair”.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives & shakes hands with each child (Hello! How are you?).
  • Remember time — cumulative review of the whole year with A4 flashcards / grab game / app on the board.
  • Focus time — presentation from the magic bus-box, topic song, board app, active games.
  • Table time 🪑 — own folder worksheet, a different type each lesson (listen & tick / colour / match / draw & tell / put-in-order / mini-craft), always language-linked (listen → do → say).
  • Reading time 🟢 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today's language, so the lesson ends on a high note.
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
