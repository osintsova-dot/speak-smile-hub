🦝 GenkiEnglish Preschool · Challenge · Lesson 23 — Baby Rabbit Family — This is my…
Genki preschoolers (~5–6 y.o., non-readers) · 60 min · songs + TPR + games (no reading / writing / phonics) · chairs · carpet · tables · board (Genki app)
Challenge level · Lesson 23 of 8✨ family · This is my…12 pupils · whole-group + shoulder-partner🎧 Bilingual L2/L1 · by ear
Aims & objectives
  • Main aim. By the end, pupils will be better able to name 7 family members and introduce them — This is my mum! This is my baby!
  • Subsidiary. Cumulative Remember-time recycle of animals, pets, voices & likes; the S&S big-book; a stick-the-family photo-frame craft.
  • Personal. Bilingual L2/L1; family always in context (This is my…, not a list); every child introduces to a partner every round; ICQ before each game; alternate ⚡/🧘.
By the end, pupils can…
  • name dad, mum, sister, brother, grandma, grandpa, baby;
  • say This is my…!;
  • run to the right family card;
  • introduce a puppet to a friend;
  • stick & name a family in a photo frame.

Classroom management — signals
“Class? — Yes!” — you: “Class!” → children stop & echo “Yes!”
Volume control 👋 — raise arm = louder, lower = quieter → instant settler.
“Speaky says…” — do the action only when Speaky the raccoon says it.
“Freeze!” / statue — everyone becomes a statue, mid-movement reset.
Countdown — “5… 4… 3… 2… 1…” with fingers shown.
“Come with me!” 🚊 — line up as a train (паровозик) to change zone.
Lesson framework — Oksana’s real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Finale = cumulative Q&A chain + the Thank You! song ritual.

  • Opening time 🚪 — door greeting (train), Hello / Good Morning song, “choose the greeting → jump/run/swim/fly to your chair”.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives on the bus-box & shakes hands with each child.
  • Remember time — cumulative Q&A of everything so far (thumb-up/down like a game), growing lesson by lesson, woven into today’s topic.
  • Focus time — presentation from the magic bus-box, Genki topic song, board app, active games.
  • Table time 🪑 — own folder worksheets, a different type each lesson (draw & tell / mini-craft / listen & tick / match соедини / spinner / sort & stick / count), always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Remember time 2 + Closing — run the growing Q&A chain, Thank You! + Goodbye song, exit ticket + reward sticker on the folder.

Cumulative Q&A — fast, like a game (thumb up/down by each picture, all 12 answer chorally, then RallyRobin round the circle):

  • Greetings: Hello! What’s your name? — My name is…! How are you? — I’m happy!

  • Do you like…?: Do you like apples? — Yes, I do! / No, I don’t!
  • I like animals: I like kangaroos / bears / giraffes / penguins!
  • I like / don’t like veggies: I like carrots! I don’t like onions!
  • Do you have any pets?: Do you have any pets? — Yes! I have a hamster / rabbit / turtle / goldfish / lizard.
  • Animal voices: What does a dog say? — Woof! (cat—meow, cow—moo, duck—quack, sheep—baa)

Wrap into today: Do you have any pets? — Yes! And this is my family — this is my mum!
