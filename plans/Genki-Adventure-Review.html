Aims & objectives
  • Main aim. By the end, pupils will have consolidated the whole Adventure level — greetings & name, feelings, I can…, left & right, pronouns with happy, and polite words — and demonstrated them with confidence and joy.
  • Subsidiary. song medley of the level; team review quiz with rewards (Banana Tree — review only); review-medley worksheet; achievement check + certificates.
  • Personal. Bilingual L2/L1; celebrate every child; whole-team answers so all 12 speak; ICQ before each game; alternate ⚡/🧘; end the level on a high.
By the end, pupils can…
  • give their name and greet;
  • ask/answer How are you?;
  • say what they can do;
  • respond to Left! Right!;
  • use he/she/we with happy;
  • use please / thank you / you’re welcome.

Classroom management — signals
“Class? — Yes!” — you: “Class!” → children stop & echo “Yes!”
Volume control 👋 — raise arm = louder, lower = quieter → instant settler.
“Speaky says…” — do the action only when Speaky the raccoon says it.
“Freeze!” / statue — everyone becomes a statue, mid-movement reset.
Countdown — “5… 4… 3… 2… 1…” with fingers shown.
“Come with me!” 🚊 — line up as a train (паровозик) to change zone.
Lesson framework — Oksana’s real Genki preschool stages

Practice-proven skeleton — reuse in every plan. Bilingual teacher talk (L2 English / L1 русская подсказка). Zones: chairs 🪑 / carpet 🌱 / tables 🪑 / board 📺 (Genki app). Cumulative Q&A recycle lives at the start (Remember time); the final game is on today’s language.

  • Opening time 🚪 — door greeting (train), Hello song (chorus), choose the greeting → move to your chair.
  • Warm up ⚡ — Disco Warm Up song; Speaky arrives & shakes hands with each child.
  • Remember time — cumulative review with A4 flashcards / memory / bridge game / app on the board.
  • Focus time — presentation from the magic bus-box, topic song, board app, active games.
  • Table time 🪑 — own folder worksheets, a different type each lesson (listen & tick / colour by command / trace / match / draw & tell / put-in-order / mini-craft), always language-linked (listen → do → say).
  • Reading time 🌱 — our S&S big-book (Speaky), read actively with gestures & choral refrain.
  • Final game ⚡ — back to the carpet for a fast, fun game on today’s language, ending on a high note (kids don’t want to leave).
  • Closing — Thank You! + Goodbye song, exit ticket + reward sticker on the folder.
