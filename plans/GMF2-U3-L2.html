Week 02.11–08.11 · Lesson 19✨ NEW · Has it got…? — Yes, it has / No, it hasn't~10 pupils · pairs + 2 teamsPupil's Book p.29 · AB p.27
Aims & objectives
  • Main aim. By the end, learners will be better able to ask & answer about animals: “Has it got (long hair)? — Yes, it has. / No, it hasn't.” and state “It's got / It hasn't got…”.
  • Subsidiary. Understand the zoo dialogue (CD1 53); listen & number (CD1 54); guessing game — deduce the animal from got-facts; write a guessing conversation (AB format).
  • Personal. Short answers with has/hasn't — never “Yes, it got”; keep the guessing game moving so every child asks several questions.
By the end, pupils can…
  • ask “Has it got…?” with 3+ different features;
  • answer “Yes, it has. / No, it hasn't.” accurately;
  • win/lead the animal guessing game using questions only.

Structure
Has it got (big ears)? — Yes, it has. / No, it hasn't.  ·  It's got… / It hasn't got…
Form
Has + it + got…? → Yes, it has / No, it hasn't. Statement negatives: It hasn't got a beak.
Pron. watch
hasn't /ˈhæznt/ — the t lands; weak it in Has_it_got /hæzɪtɡɒt/ said as one word.
Concept checks (CCQs) · Assumptions
“Has it got four legs? — can I answer ‘Yes, it is’?” → no — Yes, it HAS.
“I ask about hair — do you tell me the animal?” → no, just yes/no — that's the game!

Assumptions: 10 animals + It's got are active (L1). GMF1 U3 had has got receptively — the question form is the new step. Feature phrases readable on cards.
